1. Chat Matemático con Python y CustomTkinter

Este proyecto consiste en una aplicación tipo chat con interfaz gráfica (GUI) desarrollada en Python utilizando la librería CustomTkinter.
El chat permite realizar operaciones matemáticas básicas a través de mensajes escritos por el usuario.

2. Características

  - Interfaz gráfica moderna creada con CustomTkinter

  - Campo de entrada para escribir mensajes

  - Panel de chat con historial

  - Respuestas automáticas del chatbot

3. Operaciones soportadas:

 - suma a b

 - resta a b

 - multiplica a b

 - divide a b

 - potencia a b

4. Respuestas simples como “hola” y “gracias”
